# <div align="center">HSU-CHAT

# 📄 목차

# 🔍 개요
과목 별 대화방에서의 실시간 채팅 웹 어플리케이션.

# 📁 실행 방법
1. MYSQL DB 생성<br>
2. npm i<br>
3. npm start<br>
4. cmd > ipconfig > ipv4 주소 확인<br>
5. 웹 브라우저 주소창에 ipv4:3000 입력<br>
