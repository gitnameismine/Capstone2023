# HSU-CHAT
과목 별 대화방에서의 실시간 채팅 웹 어플리케이션 


# 실행 방법
1. DB 생성<br>
2. npm i<br>
3. npm start<br>
4. cmd > ipconfig > ipv4 확인<br>
5. 주소창에 ipv4:3000<br>


    
   ### 설치 라이브러리

    "express": "^4.17.1",
    "moment": "^2.29.1",
    "mysql": "^2.18.1",
    "socket.io": "^4.4.0"
